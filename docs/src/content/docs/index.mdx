## What is this?

`inspect-value` wraps [svelte-inspect-value](https://github.com/user/svelte-inspect-value) as a **Web Component** (`<inspect-value>`, `<inspect-panel>`). Styles are encapsulated in Shadow DOM — just import the script and use the element.

### Features

- **All JS types** — strings, numbers, booleans, arrays, objects, Date, RegExp, URL, Map, Set, Error, TypedArray, Promises, Functions, and more
- **6 built-in themes** — `inspect`, `drak`, `stereo`, `dark`, `default-dark`, `default-light`
- **Deep search** — filter, filter-strict, and highlight modes
- **Panel mode** — fixed-position overlay for persistent debugging
- **Zero config** — import the script, use the element
